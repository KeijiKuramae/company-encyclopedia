# 株式会社メドレー 企業研究レポート

## 🎯 私とのマッチ度評価 (総合評価: ★★★★★)

株式会社メドレーは、医療ヘルスケア領域に特化したITサービスを提供しており、貴殿の専攻・研究内容・興味関心との親和性が極めて高いと評価できます。

**理由:**
*   **医療情報学 / データサイエンスとの合致:** 貴殿の「医療情報学」「データサイエンス」の専門性は、メドレーが展開する医療機関向けSaaS（CLINICS、メドレーかかりつけ薬局支援など）や医療介護人材プラットフォーム（ジョブメドレー）におけるデータ活用、システム構築、課題解決に直結します。医療情報のデジタル化・データ分析を通じた医療現場の課題解決という貴殿の関心は、メドレーの事業の中核をなすものです。
*   **超上流・企画（PdM / PM / ITコンサル適性）の強み:** 貴殿の「超上流工程」における企画力、要件定義力は、メドレーが提供するプロダクトの新規開発や機能改善において、プロダクトマネージャー（PdM）やプロジェクトマネージャー（PM）として大いに活かせるでしょう。ユーザー課題を発見し、それをシステム要件に落とし込む能力は、SaaSビジネスを展開するメドレーにとって不可欠なスキルです。
*   **医療IT・ヘルスケア分野への強い興味:** 貴殿の「医療IT・ヘルスケア分野」への強い興味は、メドレーの企業ビジョン「医療ヘルスケアの未来をつくる」と完全に一致します。自身の専門性と情熱を、社会貢献性の高い事業で発揮できる環境です。
*   **技術の全体感の理解:** 実装は得意ではないとしつつも、Pythonを用いたWebスクレイピングや生成AI（LLM）の仕組みを理解している点は、技術的な実現可能性を考慮した上流工程の設計において強みとなります。

**入社後に活かせる強み:**
*   医療情報学とデータサイエンスの専門知識を活かし、医療現場の課題をデータに基づき分析し、具体的なITソリューションの企画・要件定義に貢献できる。
*   「リアルカタン」の企画・要件定義で培った超上流工程の経験を、メドレーのプロダクト開発におけるPdM/PMとして発揮し、ゼロからのサービス構想や既存サービスの改善を推進できる。
*   フィジーク競技で培った高い目標に対する継続力と実行力は、長期的な視点でのプロダクト成長や、困難なプロジェクト推進において強みとなる。

## 1. 企業概要

株式会社メドレーは、「医療ヘルスケアの未来をつくる」をミッションに掲げ、医療ヘルスケア領域に特化したインターネットサービスを提供する企業です。テクノロジーを活用し、医療現場の非効率を解消し、患者・医療従事者双方にとってより良い医療体験の実現を目指しています。

**主要な事業・プロダクト:**
*   **医療プラットフォーム事業:**
    *   **CLINICS (クリニクス):** オンライン診療・服薬指導システム、クラウド型電子カルテ、予約システムなど、医療機関の業務効率化と患者利便性向上を支援するSaaSプロダクト群。
    *   **メドレーかかりつけ薬局支援:** 薬局向けのクラウド型システムで、オンライン服薬指導や患者コミュニケーションを支援。
    *   **MEDLEY (メドレー):** 医療情報提供サービス。
*   **人材プラットフォーム事業:**
    *   **ジョブメドレー:** 医療介護分野に特化した求人サイト。医療従事者の人材不足解消に貢献。

**ビジョン:**
「医療ヘルスケアの未来をつくる」

## 2. 理系院生向け情報 (R&D・技術力)

### 研究開発(R&D)の注力分野
メドレーは、医療ヘルスケア領域におけるテクノロジー活用を推進しており、特に以下の分野に注力しています。
*   **医療データの活用と分析:** CLINICSやジョブメドレーを通じて蓄積される膨大な医療・ヘルスケアデータを活用し、医療現場の課題解決やサービス改善、新たな価値創造を目指しています。データサイエンスや機械学習を用いた分析、予測モデル構築などが含まれます。
*   **AI・機械学習の医療応用:** オンライン診療の効率化、電子カルテの入力支援、医療画像診断支援、人材マッチング精度の向上など、AI技術の医療ヘルスケア領域への応用を積極的に推進しています。
*   **SaaSプロダクト開発:** 医療機関や薬局の業務効率化、患者体験向上を目的としたクラウド型サービスの機能拡張、新規開発に継続的に投資しています。スケーラビリティ、セキュリティ、ユーザビリティの高いシステム構築が求められます。
*   **クラウド技術・マイクロサービスアーキテクチャ:** サービスの安定稼働と迅速な開発を実現するため、AWSなどのクラウドインフラを活用し、マイクロサービスアーキテクチャへの移行を進めています。

### 保有しているコア技術・強み
*   **大規模な医療介護人材データベース:** ジョブメドレーを通じて培われた、医療介護分野における国内最大級の人材データベースとマッチング技術。
*   **医療現場の深い理解に基づくプロダクト開発力:** 医師や医療従事者が経営陣にいるため、医療現場のリアルな課題やニーズを深く理解し、それに基づいた実用性の高いプロダクトを開発できる強みがあります。
*   **モダンな技術スタックの採用:** Ruby on Rails, React, TypeScript, Go, Pythonなどの言語やフレームワーク、AWSなどのクラウドサービスを積極的に採用し、技術的な挑戦を推奨する文化があります。
*   **セキュリティと信頼性へのコミットメント:** 医療情報を扱うため、高いレベルのセキュリティ対策とシステムの信頼性確保に注力しています。

### 最近の技術的なニュースや特許・論文などの動向
*   **技術ブログの公開:** メドレーは「Medley Developers Blog」を運営しており、エンジニアによる技術的な取り組みや知見が定期的に発信されています。AI/ML、インフラ、フロントエンド、バックエンドなど多岐にわたるテーマで、具体的な技術課題とその解決策が紹介されています。
*   **生成AIの活用検討:** 近年、生成AI（LLM）技術の進化に伴い、医療現場での活用可能性について積極的に情報収集・検討を進めていることが、技術ブログや採用イベントなどで示唆されています。例えば、電子カルテ入力支援や医療情報検索の効率化などが考えられます。
*   **特許・論文:** 公式に公開されている特許や論文は、一般的なIT企業と比較して多くはありませんが、内部での研究開発は活発に行われています。特に医療情報システムやマッチングアルゴリズムに関するノウハウは、企業秘密として保護されている可能性があります。

## 4. キャリアパス

### 理系出身者の主な配属先・職種
貴殿のような医療情報学・データサイエンスのバックグラウンドを持つ理系院生は、主に以下の職種での活躍が期待されます。
*   **プロダクトマネージャー (PdM):** ユーザー課題の特定、新機能の企画立案、要件定義、開発ロードマップの策定、開発チームとの連携など、プロダクト全体の責任を担う。貴殿の「超上流・企画」の強みが最も活かせる職種です。
*   **データサイエンティスト / データアナリスト:** 医療ヘルスケアデータの収集・分析、機械学習モデルの構築、サービス改善のためのインサイト抽出、データに基づいた意思決定支援など。
*   **ソフトウェアエンジニア (バックエンド/フロントエンド/SRE):** 医療SaaSプロダクトや人材プラットフォームの開発・運用。特に、医療ドメインの知識を持つエンジニアは重宝されます。
*   **プロジェクトマネージャー (PM):** 大規模な開発プロジェクトの進行管理、関係者間の調整、品質管理など。

### 採用実績（大学院など）
メドレーは、特定の大学に偏らず、幅広い大学・大学院から優秀な人材を採用しています。特に、情報科学、コンピュータサイエンス、データサイエンス、医療情報学などの分野を専攻する大学院生は、積極的に採用対象となっています。具体的な大学院名は公開されていませんが、難関国立大学や有名私立大学の大学院からの採用実績は多数あります。

### 入社後のキャリアイメージ、研修制度
*   **入社後のキャリアイメージ:**
    *   **専門性の深化:** 特定の技術領域やプロダクト領域において専門性を深め、テックリードやスペシャリストとして活躍。
    *   **マネジメント:** PdMやPMとしてプロダクトやプロジェクトを牽引するリーダー、あるいはエンジニアリングマネージャーとして組織をマネジメントするキャリアパス。
    *   **事業開発:** 技術的知見を活かし、新規事業の企画・立ち上げに参画する可能性もあります。
    *   メドレーは、個人の志向と能力に応じて多様なキャリアパスを支援する文化があります。
*   **研修制度:**
    *   **新卒研修:** 入社後、社会人としての基礎や企業文化、事業内容に関する研修があります。
    *   **OJT (On-the-Job Training):** 配属後は、OJTを通じて実務経験を積みながらスキルを習得します。
    *   **技術研修・勉強会:** 社内での勉強会や、外部のセミナー・カンファレンスへの参加を奨励・支援しています。
    *   **書籍購入補助:** 業務に必要な書籍の購入費用を補助する制度があります。

## 5. インターンシップ情報（院進予定者向け・持ち越し可否）

### 学部4年/修士進学予定者が参加できるインターンシップの有無
株式会社メドレーは、毎年夏季を中心に、学部生・大学院生を対象としたインターンシップを実施しています。特にエンジニア職、プロダクトマネージャー職のインターンシップは人気が高く、技術的なスキルやプロダクトへの熱意を重視する傾向にあります。

*   **実施時期:** 主に夏季（8月〜9月頃）に開催されることが多いですが、通年で募集している職種や、短期・長期のプログラムが存在する可能性もあります。
*   **対象:** 学部3年生〜大学院生（修士1年生、博士課程の方も含む）が主な対象となります。

### 「インターンで得た評価や早期選考パスを、大学院卒業時（30卒）まで持ち越せるか（または通年採用か）」に関する情報・クチコミ
*   **早期選考パスの持ち越し:** メドレーのインターンシップにおいて、明確に「大学院卒業時まで早期選考パスを持ち越せる」と公式に明言されている情報は見当たりません。しかし、優秀なインターン生に対しては、早期選考の案内や、次年度以降の採用活動における優遇措置が取られるケースは一般的に存在します。個別の状況や年度によって対応が異なる可能性があるため、インターンシップ参加時に直接採用担当者に確認することが最も確実です。
*   **通年採用:** メドレーは、新卒採用において通年採用の形式を明示しているわけではありませんが、中途採用は活発であり、優秀な人材であれば時期を問わず採用を検討する柔軟性を持っている可能性があります。新卒採用においても、インターンシップ経由での早期接触や、個別の状況に応じた選考が行われる可能性はあります。

### 特に「要件定義」「企画」「ITコンサルティング」「PdM」などの上流工程を体験できるプログラムの詳細
メドレーのインターンシップでは、貴殿の強みである「要件定義」「企画」「PdM」といった上流工程を体験できるプログラムが提供されることがあります。

*   **プロダクトマネージャー (PdM) インターンシップ:**
    *   **内容:** 実際のプロダクト課題をテーマに、ユーザーリサーチ、課題特定、新機能の企画立案、要件定義、プロトタイプ作成、開発チームへの提案といった一連のPdM業務を体験します。メンターとなる現役PdMの指導のもと、市場分析からプロダクトの価値提案まで、超上流工程のプロセスを実践的に学ぶことができます。
    *   **特徴:** 医療ヘルスケア領域の専門知識と、ビジネス・テクノロジー・デザインの視点を統合したプロダクト開発の難しさ、面白さを体感できます。データに基づいた意思決定の重要性も学ぶ機会となるでしょう。
*   **エンジニアインターンシップ (企画・設計要素含む):**
    *   **内容:** 純粋なコーディングだけでなく、開発テーマによっては、要件定義や設計フェーズから関与できるプログラムもあります。ユーザーの課題を理解し、技術的な実現可能性を考慮しながら、どのようなシステムを構築すべきかを検討する機会が得られます。
    *   **特徴:** 企画段階からエンジニアが参加することで、より現実的かつ効果的なソリューションを導き出すプロセスを体験できます。

これらのインターンシップは、貴殿の「超上流・企画」の強みを存分に発揮し、メドレーの事業への理解を深める上で非常に有益な機会となるでしょう。募集要項や過去の開催実績を詳細に確認し、積極的に応募することをお勧めします。
